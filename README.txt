Онлайн-каталог сотрудников.

1.Веб-страница, которая выводит иерархию сотрудников в древовидной форме (раскрывающийся список-дерево). 
● Информация о каждом сотруднике хранится в базе данных и содержит следующие данные:
  -ФИО;
  -Должность;
  -Дата приема на работу;
  -Размер заработной платы.
● У каждого сотрудника есть 1 начальник;
● База данных содержит 5 уровней иерархий.

2. Веб-страница, которая выводит список сотрудников в табличном виде:
● Выводится вся имеющаяся о сотруднике информация из базы данных;
● Реализована возможность сортировать по любому полю.


Стек: Flask, Flask-SQLAlchemy, flask_seeder, Twitter Bootstrap.
