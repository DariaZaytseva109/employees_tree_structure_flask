Онлайн-каталог сотрудников.

1. Реализована веб-страница, которая выводит иерархию сотрудников в древовидной форме (раскрывающийся список-дерево). 
2. Информация о каждом сотруднике хранится в базе данных и содержит следующие данные:	
	-ФИО;
	-Должность;
	-Дата приема на работу;
	-Размер заработной платы.
  ● У каждого сотрудника есть 1 начальник;
  ● База данных содержит 5 уровней иерархий.
3. База заполняется тестовыми данными с помощью seeder. 
4. Реализована веб-страница, которая выводит список сотрудников в табличном виде:
  ● Выводится вся имеющаяся о сотруднике информация из базы данных;
  ● Реализована возможность сортировать по любому полю.


Стек: Flask, Flask-SQLAlchemy, flask_seeder, Twitter Bootstrap, Postman, SQLite.


Стартовая страница: /employees/v1/
Требуется запустить файл main.py
Необходимые пакеты в файле requirements.txt
